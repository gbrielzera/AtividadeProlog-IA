# Atividade da disciplina de Inteligência Artificial em Prolog.


A atividade teve o objetivo de resolver um problema lógico com diferentes categorias: nome, mochila, mês, jogo, matéria e sucos).


Primeiramente as informações foram organizadas em uma tabela usando as pistas do enunciado até chegar à solução.


Depois a mesma lógica foi implementada no Prolog:


-Os elementos foram modelados como estruturas

-As restrições foram "traduzidas" em regras do prolog

-Foram utilizadas nth1 como posição, nextto como vizinhança e ordem como "logo a esquerda"

-O Prolog encontrou automaticamente a solução com backtracking


Por fim, foi criada uma função para exibir os resultados numa tabela.
